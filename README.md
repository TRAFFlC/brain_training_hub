# 🧠 脑力训练外挂 - Brain Training Hub

一个基于 Python + Tkinter 的脑力训练工具集，提供 15 个自动化小工具，帮助你提升记忆力、反应力、感知力和认知灵活性。

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)

## ✨ 特性

- 🎯 **15 个工具** - 覆盖视觉、记忆、感知、反应、认知五大类训练
- 🚀 **一键启动** - 统一管理面板，点击即可启动任意工具
- 📊 **状态监控** - 实时显示工具运行状态
- 🎨 **现代化界面** - 分类管理，美观易用
- 🔧 **高度可配置** - 支持快捷键、自定义参数
- 📦 **开箱即用** - 提供完整模板和配置文件

## 🚀 快速开始

### 1. 环境要求

- **操作系统**: Windows 10/11
- **Python**: 3.8 或更高版本
- **屏幕分辨率**: 建议 1920x1080 或更高

### 2. 安装依赖

```bash
# 克隆或下载项目后，进入项目目录
cd brain_training_hub

# 安装依赖
pip install -r requirements_v2.txt
```

**依赖说明**:
- `pyautogui` - 自动化控制和截图
- `keyboard` - 全局键盘钩子
- `Pillow` - 图像处理
- `opencv-python` - 计算机视觉
- `numpy` - 数值计算
- `mss` - 快速截图
- `imagehash` - 图像相似度匹配
- `rapidocr-onnxruntime` - OCR 识别（舒尔特 OCR 工具需要）

### 3. 启动方式

#### 方法一：双击启动脚本（推荐）
```
双击：启动脑力训练中心.bat
```

#### 方法二：命令行启动
```bash
python brain_training_hub.py
```

### 4. 使用步骤

1. **启动主面板** - 运行启动脚本，打开统一管理界面
2. **选择工具** - 在对应分类标签页下找到需要的工具
3. **启动工具** - 点击工具卡片上的"🚀 启动工具"按钮
4. **设置区域** - 部分工具需要按快捷键（F9/F10 等）设置屏幕区域
5. **开始训练** - 按照各工具的提示进行操作

## 📖 详细使用说明

### 舒尔特方格（模板匹配版）

**用途**: 自动识别并点击舒尔特方格中的数字（1→2→3...）

**使用步骤**:
1. 打开工具，将鼠标移动到游戏区域左上角
2. 按 `F9` 记录左上角坐标
3. 将鼠标移动到游戏区域右下角，按 `F10` 记录右下角坐标
4. 按 `F12` 设置开始按钮位置（如果需要自动点击开始）
5. 按 `S` 键开始一键通关

**配置保存**: 坐标会自动保存到 `config.json`

---

### 舒尔特方格（OCR 版）

**用途**: 使用 OCR 技术识别数字，无需模板，适应性更强

**使用步骤**:
1. 首次使用建议点击"预加载 OCR"下载模型
2. 按 `F9` 和 `F10` 设置游戏区域
3. 可调节方格大小（3x3 到 10x10）和点击间隔
4. 按 `S` 键或点击"一键通关"开始

**特点**:
- 无需模板，适应不同字体和样式
- 支持调试模式，保存识别截图
- 识别速度略慢于模板匹配版

---

### 找不同的字

**用途**: 在 10x10 网格中自动找到不同的字

**使用步骤**:
1. 按 `F9` 设置左上角，`F10` 设置右下角，`F11` 设置右上角目标字区域
2. 按 `S` 预览识别效果
3. 按 `W` 开始自动模式

**原理**: 使用 pHash 图像相似度算法，将网格中的每个字与目标字比对

---

### 找不同的字 - 暴力版

**用途**: 无需识别，暴力点击所有格子

**使用步骤**:
1. 按 `F9/F10/F11` 设置游戏区域
2. 调节 CPS（每秒点击次数）
3. 按 `W` 开始暴力点击

**适用场景**: 当普通版识别效果不佳时，暴力版更可靠

---

### 反应速度测试

**用途**: 测试你的反应速度，冲击排行榜

**使用步骤**:
1. 按 `F5` 设置检测点（屏幕变色检测点）
2. 按 `F7` 设置点击位置
3. 按 `F8` 开始测试
4. 工具会自动检测红变绿并点击

**技巧**: 使用窗口 DC 读取技术，无需截图，检测速度极快

---

### 反应速度 - 挂机刷分

**用途**: 自动循环点击，挂机刷极限成绩

**使用步骤**:
1. 按 `F7` 设置点击位置（游戏按钮位置）
2. 设置点击间隔（默认 1 秒，游戏限制最小 1 秒）
3. 按 `F8` 开始挂机，`F9` 停止

**原理**: 游戏随机等待≥1 秒后红变绿，挂机每隔 1 秒点一次，总会碰到正好变色的时候

**统计**: 显示点击次数、挂机时间、平均每分钟点击次数

---

### 时间感知训练

**用途**: 训练精确的时间间隔感知能力

**使用步骤**:
1. 按 `F7` 设置按钮位置
2. 按 `F8` 开始训练
3. 工具会在精确时间间隔后自动点击

**精度调整**: 详见下方"精度校准"章节

---

### 数字记忆 / 顺序记忆 / N-Back 记忆

**用途**: 训练工作记忆和序列记忆能力

**使用步骤**:
1. 按 `F9/F10` 设置游戏区域
2. 按 `S` 开始游戏
3. 工具会自动识别并记忆位置

---

### 色差感知 / 动态色差感知

**用途**: 训练颜色敏感度

**使用步骤**:
1. 按 `F9/F10` 设置游戏区域
2. 按 `S` 开始游戏
3. 工具会自动识别颜色差异并点击

---

### 斯特鲁普效应

**用途**: 训练认知灵活性

**使用步骤**:

## ⚙️ 配置说明

### 配置文件

工具会自动生成配置文件，保存在项目根目录：

| 配置文件 | 用途 |
|---------|------|
| `config.json` | 舒尔特方格区域设置 |
| `reaction_*.json` | 反应速度测试配置 |
| `time_trainer_config.json` | 时间感知训练配置 |

### 精度校准（时间感知）

时间感知工具的精度取决于系统延迟，建议定期校准：

```bash
# 运行校准工具
python calibrate_timer.py
```

**校准步骤**:
1. 点击"开始校准"，自动测试 20 次
2. 查看统计结果和建议值
3. 修改 `time_perception_trainer.py` 中的 `CLICK_LATENCY` 值
4. 重新运行训练器测试

**调整原则**:
- 实际时间 > 设定时间 → 增大 CLICK_LATENCY
- 实际时间 < 设定时间 → 减小 CLICK_LATENCY

**参考值**:
- 高性能游戏本：0.0010 ~ 0.0020
- 普通台式机：0.0015 ~ 0.0025
- 笔记本（节能模式）：0.0020 ~ 0.0035

## ⚠️ 注意事项

### 1. 权限要求

部分工具需要**管理员权限**才能正常使用键盘钩子和鼠标控制功能。建议：
- 右键点击 `启动脑力训练中心.bat`
- 选择"以管理员身份运行"

### 2. 防病毒软件

工具使用键盘钩子和鼠标自动化，可能被防病毒软件误报。请：
- 将整个项目文件夹添加到信任列表
- 或临时关闭防病毒软件

### 3. 屏幕分辨率

- **最低要求**: 1366x768
- **推荐**: 1920x1080 或更高
- 部分工具可能需要在 100% 缩放比例下运行

### 4. 性能优化

- 关闭不必要的后台程序
- 设置电源模式为"高性能"
- 禁用 CPU 节能选项（如需极限精度）

### 5. 使用建议

- **循序渐进**: 从简单难度开始
- **持之以恒**: 每天训练 15-30 分钟
- **多样训练**: 交替使用不同工具
- **适当休息**: 避免过度训练

## 🔧 故障排除

### 工具无法启动

1. 检查 Python 是否正确安装（3.8+）
2. 确认依赖库已安装：`pip install -r requirements_v2.txt`
3. 查看命令行输出的错误信息

### 工具窗口显示异常

1. 调整屏幕分辨率和缩放比例
2. 关闭其他置顶窗口
3. 重启工具面板

### 工具控制不响应

1. 以管理员身份运行
2. 关闭并重新启动该工具
3. 检查快捷键是否被其他程序占用

### 识别准确率低

**舒尔特方格**:
- 确保游戏区域设置准确
- 尝试使用 OCR 版本
- 检查屏幕亮度和对比度

**找不同的字**:
- 确保目标字区域设置正确
- 调节 pHash 相似度阈值
- 尝试暴力版本

### 时间感知精度不够

1. 运行校准工具 `calibrate_timer.py`
2. 根据建议调整 `CLICK_LATENCY`
3. 关闭后台程序，设置高性能模式
